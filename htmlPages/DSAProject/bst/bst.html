A Binary Search Tree (BST) is a binary tree in which each node has a value greater than all values in its left subtree and less than all values in its right subtree.

What is a Binary Search Tree?

A Binary Search Tree (BST) is a node-based binary tree data structure with the following properties:

  • Left Subtree: All values less than the current node.
  • Right Subtree: All values greater than the current node.
  • No Duplicate Values (in classic BSTs).

Supported Operations & Algorithms

  • Insert
  • Delete
  • Search
  • Update
  • Find Min/Max
  • Find Height
  • Inorder/Preorder/Postorder Traversal
  • Level Order Traversal (BFS)
  • Clear
