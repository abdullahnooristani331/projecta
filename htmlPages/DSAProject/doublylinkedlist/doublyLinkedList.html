What is a Doubly Linked List?

A doubly linked list is a collection of nodes where each node contains a value, a reference to the next node, and a reference to the previous node. It allows efficient insertions and deletions from both ends and supports bidirectional traversal.

  • Dynamic Size: Grows and shrinks as needed.
  • Node Structure: Each node has a value, a pointer to the next node, and a pointer to the previous node.
  • Head/Tail: The first and last nodes in the list.
  • Operations: Insert (head, tail, index), Delete (head, tail, index), Search, Update, Traverse (forward/backward), Clear

Big O Complexity Table

Operation Description Time Complexity
Insert at Head Insert node at beginning O(1)
Insert at Tail Insert node at end O(1)
Insert at Index Insert node at position O(n)
Delete at Head Remove first node O(1)
Delete at Tail Remove last node O(1)
Delete at Index Remove node at position O(n)
Search Find node by value O(n)
Update Update value at index O(n)
Traverse (Forward/Backward) Visit all nodes O(n)
Clear Remove all nodes O(1)
