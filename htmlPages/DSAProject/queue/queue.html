What is a Queue?

A queue is a linear data structure where elements are inserted from one end (the rear) and removed from the other (the front). It is used in scheduling, buffering, and breadth-first search algorithms.

  • FIFO Principle: First-In-First-Out.
  • Enqueue: Add an element to the rear.
  • Dequeue: Remove the front element.
  • Front/Peek: View the front element without removing it.
  • IsEmpty/IsFull: Check queue status.
  • Traverse: Visit all elements in order.
  • Clear: Remove all elements.

Big O Complexity Table

Operation Description Time Complexity

Dequeue Remove front element O(1)
Front/Peek View front element O(1)
IsEmpty Check if queue is empty O(1)
IsFull Check if queue is full O(1)
Traverse Visit all elements O(n)
