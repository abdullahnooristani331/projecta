What is a Stack?

A stack is a linear data structure where elements are added and removed from only one end, called the top. It is widely used for function call management, undo operations, and parsing expressions.

  • LIFO Principle: Last-In-First-Out.
  • Push: Add an element to the top.
  • Pop: Remove the top element.
  • Peek: View the top element without removing it.
  • IsEmpty/IsFull: Check stack status.

Big O Complexity Table

Operation Description Time Complexity
Push Add element to top O(1)
Pop Remove top element O(1)
Peek View top element O(1)
IsEmpty Check if stack is empty O(1)
IsFull Check if stack is full O(1)
Traverse Visit all elements O(n)
Clear Remove all elements O(n)
