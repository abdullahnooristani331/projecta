Arrays are fixed-size, contiguous data structures that allow constant-time access by index and efficient traversal. They are the foundation of many other data structures in computer science.

What is an Array?

An array is a linear, contiguous collection of elements of the same type, stored in adjacent memory locations. Arrays are widely used in Data Structures and Algorithms (DSA) for their simplicity and efficiency in accessing elements by index.

  • Fixed Size: The size is defined at initialization and cannot change.
  • Efficient Access: Retrieve or update any element using its index in O(1) time.
  • Same Type: All elements must be of the same data type.
  • Operations: Access, Update, Insert, Delete, Traverse, Search, Sort, Reverse, Merge, Clear.

Big O Complexity Table

Operation Description Time Complexity
Access Get value at index i O(1)
Update Set value at index i O(1)
Traverse Visit all elements O(n)
Search Find value in array O(n)
Insert Insert at index (static: no shifting) O(1)
Delete Delete at index (static: no shifting) O(1)
Sort Sort non-empty elements O(n log n)
Reverse Reverse non-empty elements O(n)
Merge Merge another array into empty slots O(n)
Clear Set all elements to empty O(n)
