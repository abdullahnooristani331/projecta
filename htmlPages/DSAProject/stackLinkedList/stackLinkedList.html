A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. Using a linked list, push and pop operations are performed at the head for O(1) time.

What is a Stack (Linked List)?

A stack can be implemented using a singly linked list, where the head of the list is the top of the stack. This allows efficient push and pop operations in constant time.

  • LIFO Principle: Last-In-First-Out.
  • Push: Add an element to the top (head).
  • Pop: Remove the top element (head).
  • Peek: View the top element without removing it.
  • IsEmpty: Check if the stack is empty.
  • Traverse: Visit all elements from top to bottom.
  • Clear: Remove all elements.
