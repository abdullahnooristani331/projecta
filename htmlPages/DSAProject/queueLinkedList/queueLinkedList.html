A queue is a linear data structure that follows the First-In-First-Out (FIFO) principle. Using a linked list, enqueue is at the tail and dequeue is at the head, both O(1).

What is a Queue (Linked List)?

A queue can be implemented using a singly linked list, where the head of the list is the front of the queue and the tail is the rear. This allows efficient enqueue (add rear) and dequeue (remove front) operations in constant time.

  • FIFO Principle: First-In-First-Out.
  • Enqueue: Add an element to the rear (tail).
  • Dequeue: Remove the front element (head).
  • Peek: View the front element without removing it.
  • IsEmpty: Check if the queue is empty.
  • Traverse: Visit all elements from front to rear.
  • Clear: Remove all elements.
