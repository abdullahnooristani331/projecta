About This App

Currency Converter is a web-based tool that allows users to convert between multiple international currencies using real-time exchange rates.

How it Works

  • 📡 It fetches live rates from the Frankfurter API.
  • 🔄 You can convert currencies instantly and view recent conversion trends via charts.
  • 📝 Conversion history is stored locally in your browser (LocalStorage).
  • 🌙 Includes light/dark theme support and clean responsive UI.
